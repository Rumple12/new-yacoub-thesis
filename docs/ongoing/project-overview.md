\# Project overview



This repository contains the implementation and documentation for the "new-yacoub" thesis plan.



The thesis focuses on a narrowed version of Yacoub's original infrastructure thesis:

\- self-hosted n8n

\- Docker and constrained runtime

\- Python middleware

\- webhook/API bridge

\- latency/resource/thermal measurements



A minimum Obid-compatible layer is included only to make the integrated system valid:

\- one workflow

\- one prompt set

\- one memory choice

\- one validation rule

\- one approval path

\- one shared contract



The project goal is to produce one small, measurable, end-to-end IoT proof of concept.
